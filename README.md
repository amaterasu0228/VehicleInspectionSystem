# VehicleInspectionSystem

Інформаційно-аналітична система автоматизації технічного огляду транспортних засобів.

## Опис проєкту

Система призначена для автоматизації процесів проведення технічного огляду транспортних засобів, ведення журналу перевірок, управління користувачами, формування аналітичної звітності та автоматичних нагадувань.

## Основні можливості

* авторизація та автентифікація користувачів;
* рольова модель доступу;
* ведення журналу технічних оглядів;
* онлайн-запис на технічний огляд;
* імпорт даних з Excel;
* управління документами;
* формування PDF-звітів;
* аналітична панель керівника;
* система автоматичних нагадувань.

## Технології

* ASP.NET Core
* Entity Framework Core
* SQL Server
* JWT Authentication
* API Gateway
* Microservices Architecture
